Веб сервисы:
для того чтобы проверить что сервер вообще поднялся заходим по ссылке:
http://localhost:8282/WS_Server/TestWS?WSDL
где
localhost:8282 -- сайт и порт сервера приложений (GlassFish, Tomcat)
WS_Server  -- название проэкта
TestWS  -- название веб-сервиса

url для тестирования:
http://localhost:8282/WS_Server/TestWS?Tester
там можно ввести значения и посмотреть результат выполнения (ответ)
