Создаем maven проект.
"Создать Item" ->
В поле "Enter an item name" вводим название проекта. Например TryTest.
Выбираем пункт "Создать проект maven"
И клацаем по кнопке "OK".

Создаеться проект. И переходит в основные (General) настройки проекта.
В блоке "Управление исходным кодом" выбираем Git
В поле Repository URL вводим http проект с GitHub. Например:
	http://local_github_url/r/~user/TryTest.git

Триггеры сборки
Если нужно чтобы проект запускался по расписанию в блоке "Триггеры сборки"
ставил галочку "Собирать периодически". И вводим расписание. 
Если необходимо чтобы проверка запускалась как только коммит загрузиться,
выбираем пункт "Опрашивать SCM об изменениях" и указываем расписание как часто опрашивать.

Сборка
Goals and options -- указываем что-то не стандартное, если необходимо. clean install test  писать не обязательно, но можно.

Настройки сборки
Если нужно чтобы шла рассылка -- ставим галочку напротив: "E-mail Notification".
И указываем почту кому слать. А также о чем именно сообщать.
